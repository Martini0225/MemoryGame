記憶翻牌

<img width="648" height="694" alt="image" src="https://github.com/user-attachments/assets/1f9e52bb-cc53-4d73-8998-0f389f3a4917" />

遊戲剛開始時會先預覽所有牌的位置3秒，再蓋上

<img width="641" height="684" alt="image" src="https://github.com/user-attachments/assets/b85cd85d-0535-40e0-a451-a72629a2186a" />

蓋上後即可開始點選卡牌進行配對

<img width="635" height="688" alt="image" src="https://github.com/user-attachments/assets/c6743fd9-93cf-4f9c-8632-034dfe32b1d4" />

如果配對成功，則會出現成功音效，配對失敗就會有失敗音效
途中如果想重新遊玩，可以點選左上Game的Restart按鈕，所有牌會更換位置並重新開始遊戲，如果全部配對成功，則會詢問是否重新，點選「是」即可重來
